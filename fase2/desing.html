Proposito del sitio

Una de las principales funciones de HTML es dar al texto estructura y significado de forma que un navegador pueda mostrarlo correctamente. En este trabajo explicaremos la forma en que se puede usar HTML para estructurar una página de texto añadiendo encabezados y párrafos, enfatizando palabras, creando listas y más.La mayor parte del texto estructurado está compuesto por encabezados y párrafos, independientemente de si lees una historia, un periódico, un libro de texto, una revista, etc.
